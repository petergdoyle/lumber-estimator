# Lumber Estimator

A tool to help estimate material needs for woodworking projects.

## Setup

Requires Python 3.

```bash
make setup
```

## Usage

Place your input CSV files in the `data/` directory.

```bash
make run
```

## Structure

- `src/`: Python source code.
- `data/`: Input dimensions (CSV).
- `output/`: Generated material lists and estimates.
- `Makefile`: Scripting and automation.
